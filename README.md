

## backon.css
A modern CSS reset
- ---

## Installation
```
npm install backon.css --save

or

download backon.css

```

## Use
```
<link rel="stylesheet" href="./css/backon.css/dist/backon.min.css">

or

// use module bundler
import 'backon.css/dist/backon.min.css'

require('backon/dist/backon.min.css')

```
